WEBSITE SETUP
-------------
Header logo:
  <img src="/web/logo-horizontal.png" alt="MCRC Tax & Accounting" height="48">

Favicons (in <head>):
  <link rel="icon" type="image/png" sizes="32x32" href="/web/favicon-32.png">
  <link rel="icon" type="image/png" sizes="16x16" href="/web/favicon-16.png">
  <link rel="apple-touch-icon" sizes="180x180" href="/web/apple-touch-icon-180.png">

Tip: SVG lockups render crispest at any size, but rely on the brand
fonts (Newsreader + IBM Plex Mono). The PNG lockups above have the
type baked in, so they look identical on every device — use those
unless the fonts are installed on your site.
